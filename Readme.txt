HabitTracker — Apple Watch & iPhone Habit Tracking AppOriginal Proposal- Minimal Apple Watch habit tracker designed for fast, one-tap interactions.- Shows a short list of daily habits and allows users to mark them done quickly.- Clean UI and large controls to minimize watch screen time.Additional Features Implemented- Full iPhone companion app for adding, deleting, and scheduling habits.- Two-way sync between iPhone and Apple Watch using `WatchConnectivity`.- Add habits from the Apple Watch as well as the iPhone.- Time-based local notifications to remind users at scheduled times.- Persistent data storage using `UserDefaults`.- Background refresh support and a WidgetKit complication showing daily progress.- Automatic daily reset of habit completion states.How the App Works1. Create habits on the iPhone or Apple Watch (each habit has a name and optional time).2. The habit list syncs between devices.3. Tap a habit on either device to mark it completed.4. Notifications remind users at scheduled times; completed habits cancel their pending notifications.5. The watch complication shows daily progress at a glance.Technologies Used- Swift & SwiftUI- WatchConnectivity- UserNotifications- WidgetKit- UserDefaults for persistenceScreenshotsAll required screenshots for the assignment are included in the Screenshots folder.GitHub Repositoryhttps://github.com/jujhar6226/HabitTracker.git
